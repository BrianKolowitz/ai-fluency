# Capstone Projects

Capstone projects are where learning becomes real. Each project builds through a series of exercises to produce a **tangible deliverable** — something you'll actually use in your work.

Choose a project that matches your current level, or challenge yourself with one above it. Every project uses AI at each step, reinforcing the skills from the competency levels.

| Project | Difficulty | Skills Practiced | Deliverable |
|:--------|:-----------|:-----------------|:------------|
| [Expense Report Builder](./expense-report.md) | Beginner → Capable | Structured prompting, data formatting, verification | Formatted expense report ready for submission |
| [Meeting-to-Action Pipeline](./meeting-pipeline.md) | Capable → Proficient | Summarization, prompt chaining, professional communication | Reusable meeting follow-up workflow |
| [Process Documentation & Improvement](./process-improvement.md) | Proficient → Expert | Workflow analysis, AI-first redesign, documentation | Current-state SOP + AI-improved proposal |
| [Personal AI Playbook](./playbook.md) | All Levels | Meta-learning, knowledge curation, teaching others | Reusable prompt library + lessons learned |

## How Capstones Work

Each project follows the same structure:

1. **Exercises** — Step-by-step, each building on the last (10-20 min each)
2. **Checkpoints** — Verify your progress before moving on
3. **Final Deliverable** — Combine your exercise outputs into something complete
4. **Reflection** — What did you learn? What would you do differently?

The **Personal AI Playbook** is the universal capstone — everyone builds one regardless of which other project they choose. It becomes your portable reference for AI-assisted work.

*These projects are designed to be completed with free AI tools (ChatGPT, Claude, Grok) and require no special access or software.*
